## ğŸ†˜ **Common Issues**

### **Issue 1: "File too large" error**

**Solution:**
```bash
# Check file sizes
du -sh *

# Remove large files from staging
git rm --cached LARGE_FILE.extension

# Add to .gitignore
echo "LARGE_FILE.extension" >> .gitignore
```

### **Issue 2: ".DS_Store everywhere"

**Solution:**
```bash
# Remove all .DS_Store files
find . -name .DS_Store -print0 | xargs -0 git rm -f --ignore-unmatch

# Already in .gitignore (âœ… done!)
```

### **Issue 3: "Accidentally pushed secrets"

**Solution:**
```bash
# Remove from git history (careful!)
git filter-branch --force --index-filter \
  "git rm --cached --ignore-unmatch SECRET_FILE" \
  --prune-empty --tag-name-filter cat -- --all

# Force push (if absolutely necessary)
git push origin --force --all
```
